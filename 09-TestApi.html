Test API

Le projet complet consistait à refaire une application mobile dans un language connue des developpeurs principaux de l'entreprise pour pouvoir dans le futur faire des mise à jour et maintenance. Grâce aux API déjà codées et à une application de low cade nomée "Convertigo", notre mission consistait à tester depuis Convertigo tout les chemins de l'API, en créant des méthodes pour chaque chemin.

Planifier les activités

En groupe de trois étudiant, notre mission consistait à tester tout les chemin d'une API déja existante dans l'application Convertigo. Convertigo étant une application de low code, pour cette mission nous n'avions pas de code à faire. Pour faciliter la comunication de ce projett, je me suis décider à créer un googledoc récapitulant toute les chemins de l'API avec un code couleur pour nous y repérer.

On peut y voir tous les chemin API triés par catégorie et un code couleur en légende pour savoir si un chemin est terminer, en cours, non fonctionnel ou pas commencé.
