Developpement web responsive

Le projet consistait à créer une application web permettant de créer et de stocker notre CV, et de l'associer à un code QR. Ce code QR restait inchangé, même si le CV était modifié : en scannant le code QR, on pouvait visualiser le nouveau CV. Ma mission était de développer l'aspect visuel des pages en utilisant une charte graphique, Bootstrap et uniquement HTML et CSS, de manière à ce qu'elles s'adaptent à différentes tailles d'écran.

Compétences

Traiter des demandes concernant les applications

L'objectif de ce projet était de concevoir les différentes pages de l'application web en utilisant HTML et CSS. Il était essentiel que ces pages soient responsive et donc qu'elles intègrent Bootstrap, ce qui nous a amenés à apprendre les bases de cet outil. Nous avions accès aux schémas des pages pour nous guider dans leur création.

Dans ce projet nous avons donc traiter des demandes concernant l'application web.
